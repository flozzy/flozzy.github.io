SCHEDULE IT

This is something I designed myself. I developed high-fidelity wireframes, clickable prototypes, and user flows for the mobile application. It is a scheduling application design. th time at hand and the pandemic we are in, it will help save time. It allows the user to schedule hospital appointments with any hospital/ Clinic near them without having to go physically to the hospital and wait. This will have the patient with time management and help reduce crowd in the hospital/clinic.

The survey for this is Here






Landing and welcoming Page

This is the page that welcomes and introduces the user to the application.

Login and Sign up in Page

This is where the user inputs they details to login or if they don't have account they have to sign up.

Home Page

This is where the user Sees the hospital appointments they have made by dates.

Booking an Appointment

Here the user inputs the appointment details. Adding a title helps the user to identify the appointment from other appointments.

Booking

The user can select any time slot. They can also input the reason for the appointment and they have to describe their symptoms they have.

Booking

They can put in the level of pain they are feeling. The picture in the middle helps the user to locate hospitals/clinics near them they would want to book with. They can also search for other hospitals/clinics that are not near them. The map shows them how many doctors are available in each hospital/clinic, according to the date and time they have selected.


The user will get notified when the appointment has been confirmed with the hospital /clinic they have booked with. The picture on the right shows all the scheduled appointments, the status and if the appointment has an alarm on.



Reschduling

The user can change the appointment time, date and even the hospital/clinic.


The user can set a reminder and an alarm.


When the appointment has been confirmed by the hospital/clinic the user will receive a notification, and the status will cahnge.


They user can cancel the appointment if the want to. The hospital/clinic will send them a notification confirming the cancelation of the appointment. and the Appointment list will update when the cancelation has been confirmed.
